# 一、MySql基础

## 1、MySql简介

### （1）什么是MySql？

* MySql软件提供了一个非常快速、多线程、多用户和健壮的Sql数据库服务器（结构化查询）。MySql服务器适用于任务关键型、高负载的生产系统以及嵌入到大规模部署的软件中。

### （2）安装

* [参考官网](<https://dev.mysql.com/doc/refman/8.0/en/installing.html>)

### （3）启动

```shell
mysql -h host -u user -p
# Enter password: ********
```

### （4）退出

```mysql
QUIT
```
